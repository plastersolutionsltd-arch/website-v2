# My Website

A modern, scalable website design project with a clean architecture.

## Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
cd my-project/website
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

See `CLAUDE.md` for detailed structure overview.

- **`src/components/`** - Reusable UI components
- **`src/pages/`** - Full page components
- **`src/styles/`** - CSS stylesheets
- **`src/utils/`** - Helper functions
- **`src/assets/`** - Images, fonts, media
- **`public/`** - Static files
- **`tests/`** - Test suites
- **`docs/`** - Documentation

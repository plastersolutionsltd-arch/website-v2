# 🖼️ Hero Images Setup Guide

Your website is now ready to display hero images for each service page! Follow these simple steps to add your images.

---

## 📁 Step 1: Create the Images Folder Structure

On your computer, navigate to your website folder and create this structure:

```
website/
├── index.html
└── images/
    └── hero/
```

**How to create folders:**
1. Open File Explorer / Finder
2. Navigate to your website folder (where index.html is)
3. Right-click → "New Folder" → Name it `images`
4. Double-click into `images` folder
5. Right-click → "New Folder" → Name it `hero`

---

Your website is looking for **6 image files** in the `images/hero/` folder:

| Page | Image Filename | What It Shows |
|------|----------------|---------------|
| **Home Page** | `silicone-service.jpg` | Full width hero for home page |
| **Silicone Service** | `silicone-service.jpg` | Silicone render project examples |
| **Monocouche Service** | `monocouche-service.jpg` | Monocouche render examples |
| **EWI Service** | `ewi-service.jpg` | External wall insulation examples |
| **Plastering Service** | `plastering-service.jpg` | Internal plaster/ceiling work |
| **Dry Lining Service** | `drylining-service.jpg` | Dry lining & boarding work |
| **Project Gallery** | `gallery-projects.jpg` | Before & after project showcase |

---

## 🎯 Step 3: Add Your Images

1. **Find your image files** on your computer (photos of your work, projects, etc.)
2. **Copy them** (right-click → Copy)
3. **Navigate to** `website/images/hero/` folder
4. **Paste them there** (right-click → Paste)
5. **Rename them** to match the filenames above exactly

**Example:**
```
If you have a photo called: IMG_1234.jpg
Rename it to: silicone-service.jpg
```

---

## 📋 Image Naming Convention

Each filename **MUST** be exactly as shown below (case-sensitive on Mac/Linux):

```
✅ CORRECT:
- silicone-service.jpg
- monocouche-service.jpg
- ewi-service.jpg
- plastering-service.jpg
- drylining-service.jpg
- gallery-projects.jpg

❌ WRONG:
- Silicone Service.jpg (spaces)
- silicone_service.jpg (underscore)
- silicone-render.jpg (different name)
- silicone-service.jpeg (wrong extension)
```

## 🖼️ Image Best Practices

### **Recommended Image Sizes:**
- **Optimal Size:** 1920 x 1080 pixels (landscape)
- **Minimum Width:** 1280 pixels
- **Height:** 1080 pixels is ideal for full-screen hero sections

### **File Format & Size:**
- **Format:** JPG (best for photos) or PNG (if you need transparency)
- **File Size:** Keep under 500KB per image for fast loading
- **Resolution:** 72 DPI is fine for web (no need for print quality)

### **Image Content Tips:**
- **Full Width:** Photo should fill the entire width of the screen
- **Quality Projects:** Show your best work - clean, professional photos
- **Bright & Clear:** Good lighting makes images pop on the hero section
- **Include People/Before-After:** Shows the impact of your work

### **Compression/Optimization:**
Before adding images, you can compress them:
1. Visit: https://tinyjpg.com or https://imagecompressor.com
2. Upload your image
3. Download the compressed version
4. Use the compressed version in your website folder

## 🔍 Troubleshooting

**Images not showing on the website?**

1. **Check file names** - Must match exactly:
   - `silicone-service.jpg` (not `silicone_service.jpg`)
   - `monocouche-service.jpg` (not `Monocouche Service.jpg`)

2. **Check file location** - Images must be in:
   - `website/images/hero/` (not just `images/` or `hero/`)

3. **Check file extension** - Must be `.jpg` or `.png` (not `.jpeg`)

4. **Reload website** - Hard refresh your browser:
   - Windows: `Ctrl + Shift + Delete`
   - Mac: `Cmd + Shift + Delete`

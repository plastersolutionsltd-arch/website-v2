## 🔧 How It Works

When someone visits your website:

1. **Page loads** with service data
2. **Hero section mounts** in React component
3. **Service heroImage property is read** (e.g., `silicone service.jpg`)
4. **Background image URL built** with correct path encoding
5. **Full-width hero image displays** with dark overlay gradient
6. **Service content overlays** the image (title, buttons, etc.)
7. **Responsive design** adapts to all screen sizes
8. **Smooth scrolling** reveals content below

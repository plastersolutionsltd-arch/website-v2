## 🔧 Customization Guide

### **To Update Reviews:**

1. Find the reviews array in GoogleReviewsSection function (around line 2564)
2. Update author names, dates, ratings, and text
3. Keep verified: true for all reviews
4. Save and refresh

### **To Change GMB Link:**

1. Update in GoogleReviewsSection (line ~2566)
2. Also update in CLAUDE.md (for reference)
3. Both places will link to your GMB

### **To Change Colors:**

1. Find the color classes in Google Reviews cards
2. Update Tailwind classes:
   - Yellow stars: `text-yellow-400`
   - Lime badges: `text-lime-400`
   - Borders: `border-slate-200` → `border-lime-400`
